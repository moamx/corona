## طريقه التنصيب

1️⃣ تنزل بايثون
``` 
$ sudo apt update
$ sudo apt install python3 
```
2️⃣ تحمل المكاتب
``` 
$ pip install pyTelegramBotAPI
$ pip install requests
```
3️⃣ تحمل البوت
``` 
$ git clone https://github.com/moamx/corona.git
```
4️⃣ تدخل للمجلد الي فيه البوت
``` 
$ cd corona
```
5️⃣ تشغل البوت
``` 
$ python3 bot.py
```
